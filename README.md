# Discreet-Spiral-Generator-Python  

A python program that generates dicreet spirals such as Vogels Model based on the fermats spiral  

### Core equations  

### r = c*√n  
### θ = (360/α)*n  

### Dependencies  

[matplotlib](https://pypi.org/project/matplotlib/)
